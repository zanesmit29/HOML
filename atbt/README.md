# Automate the Boring Stuff with Python – Project Portfolio

Welcome! This repository showcases projects I’ve completed following the concepts and exercises from the *Automate the Boring Stuff with Python* handbook by Al Sweigart. These projects demonstrate practical Python programming skills—especially focused on automating everyday tedious tasks.

## 📚 About the Projects

Each project in this repo is designed to streamline repetitive, “boring” computer tasks. You’ll find scripts for:
- File management and organization
- Data extraction and processing (text, PDF, Excel, CSV)
- Web scraping and automation

## 🛠 Project List

| Project Name              | Description                                                         |
|---------------------------|---------------------------------------------------------------------|

| Excel Data Automation     | Program to copy and process data in Excel sheets automatically.     |

## 🚀 How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/automate-boring-stuff-projects.git
   cd automate-boring-stuff-projects
   ```

2. **Install dependencies**  
   Most scripts use only Python’s standard library, but some may need additional modules. To install them:
   ```bash
   pip install -r requirements.txt
   ```
   *(Each project’s folder contains details and dependencies in its README or header comments.)*

3. **Use the scripts**  
   Follow instructions in each project’s README or script docstring. Most can be run from command line, for example:
   ```bash
   python project_folder/file_rename.py
   ```
